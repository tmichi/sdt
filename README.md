# sdt
An implementation of distance fields on a unit sphere

![Snapshot](result.png)

# Libraries that I use
   - OpenGL + glut
   - glui
   - vecmath package C++ port : http://objectclub.jp/download/vecmath_e 

# build 
We confirmed the codes can be built on CMake + Mac.
